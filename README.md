# 🌐 WebPro - Diseño Web Profesional

¡Bienvenido al repositorio oficial de **WebPro**! Este proyecto es una landing page moderna, rápida y de alto impacto diseñada para ofrecer servicios de creación de páginas web para negocios, profesionales y empresas.

---

## 🎨 Características Principales

- **Diseño Premium & Moderno**: Una estética cuidada inspirada en las últimas tendencias de UX/UI.
- **Modo Oscuro/Claro**: Interruptor dinámico de tema con persistencia mediante localStorage.
- **Optimizado para Móviles**: Diseño totalmente responsivo que se ve increíble en cualquier dispositivo.
- **Integración con WhatsApp**: Botones de contacto directo y código QR personalizado para una comunicación instantánea.
- **Rendimiento Extremo**: Construido con Vite y Vanilla JS, logrando tiempos de carga ultrarrápidos.
- **SEO Ready**: Configuración completa de metadatos Open Graph, Twitter Cards, Sitemap y robots.txt para una indexación perfecta en Google.

---

## 🛠️ Tecnologías Utilizadas

- **Núcleo**: HTML5 semántico y Javascript moderno (ES6+).
- **Estilos**: CSS3 moderno (Custom Properties, Flexbox, Grid, Glassmorphism).
- **Iconografía**: [Lucide Icons](https://lucide.dev/).
- **Tipografía**: [Google Fonts](https://fonts.google.com/) (Outfit & Inter).
- **Herramienta de Construcción**: [Vite](https://vitejs.dev/).
- **Despliegue**: [Vercel](https://vercel.com/).

## 🚀 Instalación y Desarrollo Local

Si deseas clonar este proyecto y ejecutarlo localmente:

1. **Clonar el repositorio**:
   ```bash
   git clone https://github.com/frandev25-gif/WebPro.git
   ```

2. **Entrar al directorio**:
   ```bash
   cd WebPro
   ```

3. **Instalar dependencias**:
   ```bash
   npm install
   ```

4. **Ejecutar servidor de desarrollo**:
   ```bash
   npm run dev
   ```

5. **Construir para producción**:
   ```bash
   npm run build
   ```
